# Open Sans Fonts

Download Open Sans from Google Fonts and place the following files in this directory:

- `opensans_regular.ttf` - Regular (400)
- `opensans_medium.ttf` - Medium (500)
- `opensans_semibold.ttf` - SemiBold (600)
- `opensans_bold.ttf` - Bold (700)

Download URL: https://fonts.google.com/specimen/Open+Sans

After adding the fonts, update `Typography.kt` to use the `OpenSans` FontFamily.
